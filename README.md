# @guilhermecbatista/biome

Guilherme Batista's personal, shared Biome configuration for modern web development.

## Installation

You can install this package via your preferred package manager. It requires `@biomejs/biome` as a peer dependency.

```bash
# Using bun
bun add -d @guilhermecbatista/biome @biomejs/biome

# Using pnpm
pnpm add -D @guilhermecbatista/biome @biomejs/biome

# Using npm
npm install --save-dev @guilhermecbatista/biome @biomejs/biome
